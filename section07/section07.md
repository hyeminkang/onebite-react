라이프사이클
ㄴ 생애주기 무언가의 탄생부터 죽음까지의 단계

Mount
ㄴ 처음 탄생한 순간 (화면에 렌더링 되는 순간 )
Update
ㄴ 다시 렌더링 되는 순간 - 리렌더링 될때
UnMount
ㄴ 컴포넌트가 화면에서 없어지는 순간 (렌더링에서 제외 되는 순간)

ex)
컴포넌트가 마운트 되었을대, 백엔드서버로 네트워크 요청을 보내서 데이터를 불러오는 기능을 만들고(mount),
컴포넌트가 업데이트 될때, 현재 값이 어떻게 변하 였는지 콘솔에 출력하는 기능을 만들고(update)
컴포넌트가 사용하던 기능을 정리(unMount)
이 과정을 "라이프 사이클 제어" 라고 함


UseEffect 
ㄴ 리액트 컴포넌트의 사이드 이펙트(부작용, 부수적인 효과, 파생되는 효과 등으로 표현가능)를 제어하는 새로운 React Hook
라이프사이클제어도 사이드이펙트라고 볼수있음
